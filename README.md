# PTSD Detection using EDAIC WOZ dataset

## Model architecture

```
┌──────────────────────────┐       ┌──────────────────────────┐     ┌──────────────────────────┐
│     Audio Input          │       │     Video Input          │     │    Audio2 Input          │
│  (seq_len, mods_size[0]) │       │  (seq_len, mods_size[1]) │     │  (seq_len, mods_size[2]) │
└─────────────┬────────────┘       └─────────────┬────────────┘     └─────────────┬────────────┘
              │                                  │                                │
┌─────────────▼─────────────┐     ┌──────────────▼────────────┐     ┌─────────────▼─────────────┐
│  Audio Projection         │     │  Video Projection         │     │  Audio2 Projection        │
│  Linear(mods_size[0]→128) │     │  Linear(mods_size[1]→128) │     │  Linear(mods_size[2]→128) │
└──────────┬────────────────┘     └──────────┬────────────────┘     └──────────┬────────────────┘
           │                                 │                                 │
┌──────────▼────────────┐          ┌─────────▼─────────────┐         ┌─────────▼─────────────┐
│ Transformer Encoder   │          │ Transformer Encoder   │         │ Transformer Encoder   │
│  Layers: 2            │          │  Layers: 2            │         │  Layers: 2            │
│  Heads: 8             │          │  Heads: 8             │         │  Heads: 8             │
│  FFN: 64              │          │  FFN: 64              │         │  FFN: 64              │
└──────────┬────────────┘          └──────────┬────────────┘         └──────────┬────────────┘
           │                                  │                                 │
┌──────────▼────────────┐          ┌──────────▼────────────┐         ┌──────────▼────────────┐
│  Mean Pooling         │          │  Mean Pooling         │         │  Mean Pooling         │
│  (seq_len→1)          │          │  (seq_len→1)          │         │  (seq_len→1)          │
└──────────┬────────────┘          └──────────┬────────────┘         └──────────┬────────────┘
           │                                  │                                 │
           │                                  │                                 │
           │                                  │                                 │
           ▼                                  ▼                                 ▼
┌────────────────────────────────────────────────────────────────────────────────────────────┐
│                                       Concatenation                                        │
│                              (128 + 128 + 128 = 384 features)                              │
└─────────────────────────────────────────────┬──────────────────────────────────────────────┘
                                              │
                                              ▼
                                   ┌─────────────────────────┐
                                   │       Classifier        │
                                   │    Linear(384→2)        │
                                   └─────────────────────────┘
```
